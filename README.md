## MadSDK
 
MADS — это омниканальная рекламная платформа, созданная для роста выручки от рекламных размещений. Инструменты рекламной платформы позволяют гибко конфигурировать медийные и performance пакеты под потребности конкретной ЦА, а также используют данные поведения пользователей, чтобы формировать для каждого пользователя персонализированный опыт.

MADS SDK — это готовая основа для интеграции ресурсов ритейлера с рекламной платформой MADS.

С помощью MADS SDK можно:

- Показывать баннеры и медийную рекламу в приложении и на сайте
- Бустить товары в поисковой, рекомендательной выдаче и листингах
- Получать аналитику по показам, кликам, конверсиям и доходности
- Отслеживать эффективность использования рекламного инвентаря ритейлера
